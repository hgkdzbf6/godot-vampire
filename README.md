# 🗡️ Vampire Sword Survivor

一个用 **Godot 4.6.1** 开发的「吸血鬼幸存者 + Roguelite + 自动攻击」游戏。所有图形均为代码自绘，**零外部素材依赖**，开箱即玩。

## 玩法核心

- 玩家只控制 **移动 / 走位**（WASD 或方向键），不主动攻击。
- 一把剑自动**围绕角色旋转**攻击敌人，剑经过的路径上造成伤害。
- 击杀敌人掉落经验，吸收经验 **升级**，每次升级 **三选一** 强化。
- 地图上随机刷新 **Buff 道具**，走过去就能吃，不需要打怪（治疗/加速/攻击/攻速/磁铁/护盾/经验）。
- 坚持 3 分钟后 **Boss 登场**，击败即胜利；血量归零即失败。

## 运行

```bash
# 用 Godot CLI 启动
godot path/to/project.godot

# 或在 Godot 编辑器中打开项目后按 F5
```

主场景：`res://src/main/Main.tscn`

## 操作

| 按键 | 功能 |
|------|------|
| `WASD` / `方向键` | 移动走位 |
| `1` / `2` / `3` | 升级时选择对应强化（也可鼠标点击） |
| `R` | 死亡/胜利后重开 |
| `Esc` / `P` | 暂停（预留） |

## 成长系统（升级三选一）

剑系统（核心成长）：

| 升级 | 效果 |
|------|------|
| 剑数量 +1 | 增加一把环绕飞剑（最高 8 把） |
| 攻击范围 +20% | 旋转半径增大 |
| 旋转速度 +25% | 转得更快，DPS 提升 |
| 剑身增大 +25% | 更大命中判定 |
| 穿透 +1 | 同时多命中一个目标 |
| 剑伤害 +30% | 单次伤害提升 |

进攻 / 元素：

| 升级 | 效果 |
|------|------|
| 暴击率 +12% | 概率 2 倍伤害 |
| 暴击伤害 +0.3x | 暴击倍率 |
| 吸血 +1 | 每次命中回 1 血 |
| 击退 +50% | 推开敌人 |
| 🔥 炽焰附魔 | 灼烧持续伤害 |
| ❄️ 寒冰附魔 | 减速敌人 |
| ⚡ 雷电附魔 | 连锁附近敌人 |

生存 / 工具：最大生命、生命回复、闪避、移速、拾取范围、完全治疗。

## 配置文件

所有游戏参数集中在 **`config/game_config.json`**，改这里即可调平衡，无需动代码：

- `player` / `sword`：玩家与剑的基础数值（剑转速、伤害、半径等）
- `enemy_types`：敌人种类表（血量/速度/伤害/经验/颜色/解锁时间）
- `boss`：Boss 血量/速度/登场时间
- `spawn`：刷怪节奏（间隔/衰减/波次）
- `xp`：经验曲线
- `pickup_buff`：Buff 道具的种类、刷新频率、效果强度
- `upgrade`：升级可选项数量

## Boss 技能（深渊领主）

Boss 使用状态机轮换 6 种技能，血量越低高威胁技能权重越高：

- **冲锋** — 朝玩家方向高速冲刺，路径留 AoE
- **扇形斩** — 前方扇形范围预警后爆发
- **地面轰炸** — 玩家脚下连续多点 AoE
- **弹幕** — 全向环形子弹 + 朝玩家扇形子弹
- **召唤** — 召唤小怪助战
- **激光扫射** — 蓄力后扫射激光（低血量触发）

所有技能都有**红色预警指示器**，给玩家走位反应时间。

## 设计要点

- **零素材**：所有角色、剑、Boss、UI、特效均用 `_draw()` + 基本图元绘制。
- **信号解耦**：`GameEvents` autoload 作为总线，模块间无硬引用。
- **数据驱动**：升级表、敌人表、常量集中在 `data/`，调平衡只改一处。
- **碰撞分层**：player=1, enemy=2, boss=8, bullet=16，剑 mask 同时检测 enemy/boss。
